Thingybase Export
=======================

Export [Thingybase](https://www.thingybase.com) data to a text file.

Usage

    export THINGYBASE_KEY=<your API key, created under profile>
    make


Dependencies

* jq  (`brew install jq`)
* cmark (`brew install cmark`)
* xmllint (provided with MacOS)
